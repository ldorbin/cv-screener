# Senior Data Engineer — Acme Fintech

We're building the data platform that powers risk, compliance, and product analytics for a regulated payments business processing £4B / year.

## What you'll do
- Own and evolve our batch + streaming data platform (Airflow, dbt, Kafka, Snowflake).
- Design data models for financial transactions that a risk analyst can self-serve from.
- Build CDC pipelines from OLTP Postgres into Snowflake with sub-minute latency for critical tables.
- Partner with analytics engineers, ML engineers, and compliance to deliver auditable data products.
- Mentor two mid-level data engineers.

## Required
- 5+ years in data engineering or similar.
- Deep SQL (window functions, performance tuning on 100M+ row tables).
- Strong Python for data (pandas, pyarrow, or equivalent).
- Production experience with Airflow (or Prefect/Dagster), dbt, and a cloud warehouse (Snowflake/BigQuery/Redshift).
- Comfortable in a regulated environment — audit trails, PII handling, change management.

## Nice to have
- Kafka or Kinesis streaming pipelines at scale.
- Fintech / regulated-industry background.
- IaC (Terraform) for data infra.
